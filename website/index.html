03
Deep dive · ATLAS

A 5‑stage pipeline,
orchestrated end-to-end

Step Functions coordinate every stage — ingest to index — with full observability through CloudWatch and X‑Ray.

01 📥 Ingest S3 → SQS → Lambda
02 👕 Rekognition Garment detection
03 🧬 CLIP Embed SageMaker ViT-L/14
04 🤖 Claude 3 Sonnet 180+ attributes
05 🔍 OpenSearch k‑NN + keyword index

Multimodal reasoning

Claude analyses the image + Rekognition labels + merchant text to understand product intent. Return risk flags are extracted at ingest, not as an afterthought.

Style embedding search

512‑dim vectors enable aesthetic search — "quiet luxury", "coastal grandmother" — not just keyword matching. Semantic distance surfaces intent.

04
Deep dive · MIRROR

Predict. Explain.
Prescribe.

Three layers that turn a return risk score into a concrete, ROI-quantified intervention — delivered in under 150ms at checkout.

01

Predict

XGBoost on 200 features across 4 domains — sizing, content, customer history, order context. Scores every order at checkout, in real time.

02

Explain

DoWhy causal graph identifies root causes, not just correlations. Attributions like "sizing: 78%", "photography: 61%" — so you fix the right thing.

03

Prescribe

Bedrock Claude generates ranked interventions with ROI estimates. "Add a size chart → 12% return reduction → $4,200/month saved."
